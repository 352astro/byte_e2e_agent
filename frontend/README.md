# Frontend

React + Vite + TypeScript. See the [project README](../README.md) for setup instructions.

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server (port 5173) |
| `npm run build` | Production build |
| `npm run lint` | ESLint |
| `npm run lint -- --fix` | ESLint + auto-fix |
| `npm run test` | Vitest |
| `npm run gen-types` | Generate types from live backend |
| `npm run gen-types:local` | Generate types from local OpenAPI spec |
